=====================================
    eBay Tools v3.0.0 - Windows Installation Package
=====================================

OVERVIEW:
eBay Tools is a comprehensive automation suite for eBay listing creation using AI/LLM technology. 
It provides photo analysis, description generation, price analysis, and CSV export capabilities.

INSTALLATION INSTRUCTIONS:
1. Ensure Python 3.8 or higher is installed
   - Download from: https://www.python.org/downloads/
   - IMPORTANT: Check "Add Python to PATH" during installation

2. Run "install.bat" to:
   - Install required dependencies
   - Create desktop shortcuts
   - Set up the application

3. Use the desktop shortcuts or batch files to launch applications

APPLICATIONS INCLUDED:
- eBay Tools - Setup: Create and manage item queues
- eBay Tools - Processor: AI-powered photo analysis and description generation
- eBay Tools - Viewer: Review results and export to eBay format
- eBay Tools - Price Analyzer: Market pricing analysis
- eBay Tools - User Manual: Complete documentation

BATCH FILES:
- install.bat: Main installation script
- ebay_setup.bat: Launch Setup Tool
- ebay_processor.bat: Launch Processor Tool
- ebay_viewer.bat: Launch Viewer Tool
- ebay_price.bat: Launch Price Analyzer
- ebay_direct_listing.bat: Launch Direct Listing Tool

WORKFLOW:
1. Setup Tool: Create item queues with photos
2. Processor Tool: Process photos with AI (enable pricing analysis)
3. Viewer Tool: Review and export to CSV for eBay upload

CONFIGURATION:
- API keys can be configured in any application's Settings menu
- Supports LLaVA (free), Claude, and OpenAI GPT-4 Vision
- All settings are saved automatically

REQUIREMENTS:
- Python 3.8 or higher
- Internet connection for AI processing
- 4GB RAM minimum (8GB recommended)
- 1GB free disk space

SUPPORT:
- Complete documentation in "eBay_Tools_User_Manual.md"
- Check logs folder for troubleshooting
- Restart applications if issues occur

For detailed instructions, open the User Manual after installation.
